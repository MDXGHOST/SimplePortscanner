# SimplePortscanner
Simple Python TCP Port Scanner with menu interface

# Description

SimplePortscanner is a Python-based TCP port scanning tool developed to understand network communication and service availability. The project demonstrates practical knowledge of TCP/IP networking, port behavior, input validation, and structured program design. It was built as a learning project to strengthen my foundation in network diagnostics and system integration concepts.

# Languages and Utilities Used

- Python
- Visual Studio Code

# Environments Used

- Windows 11
